## Glossário

Definições de termos importantes usados no ecossistema MyST.

:::{glossary}

MyST
: **Markedly Structured Text**. Uma extensão do Markdown projetada para escrita técnica e científica, totalmente compatível com Sphinx e Jupyter.

Diretiva
: Um bloco de conteúdo estruturado no MyST, delimitado por `::: {nome}` ou ` ```{nome} `. Exemplos incluem `figure`, `note`, `code-cell`.

Role
: Um comando inline no MyST, usado como `{nome}argumento`. Exemplos incluem `{ref}`, `{cite}`, `{math}`.

Frontmatter
: Metadados YAML no início de um arquivo Markdown, usados para configurar título, autores, kernelspec, e outras opções da página.

Jupyter Book
: Uma ferramenta para construir livros e documentos bonitos e com qualidade de publicação a partir de material computacional.

Thebe
: Uma biblioteca que permite transformar blocos de código estáticos em HTML em células executáveis interativas, conectando-se a um kernel Jupyter (Binder ou JupyterLite).

Pyodide
: Uma distribuição Python para o navegador e Node.js baseada em WebAssembly. Permite executar Python no lado do cliente sem servidor backend.

:::
